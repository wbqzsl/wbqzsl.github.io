Builder Pattern、Factory Pattern、Abstract Factory Pattern的区别

发表于 2020-09-26 | | 阅读次数：

Builder Pattern

Builder和抽象工厂类似，也是建立一系列对象的实现。区别在于Builder建立的一对象有依赖关系而Factory的对象只是有关联。
就像电脑builder，不仅需要建造还需要一些出厂内置设置，而Factory只管建造，造出来就不管了。

builder模式

Factory Pattern

抽象工厂不仅仅负责一个产品的生成，而是创造一类产品，如代码中的抽象快速车工厂，既可以生成小快车，也可以生成大快车，但他们都是快车
而工厂模式只是生产大车和小车，大车就是大车工厂，小车就是小车工厂。
