sfml图形库的配置

发表于 2019-05-24 | 分类于 c++ | | 阅读次数：


好久没更新了
前一段时间在学sfml图形库（c++）的使用
包括环境的配置,窗口的建立等
其简单的易懂的代码语法设计可以说是对入门新手的福音

其实对于环境的配置，官网上的说明已经很明白了

官网链接

就算官网需要翻墙，还用很多国内大佬的解释也很清楚

VS2017编译SFML SDK配制环境详解

所以我这里就补充几条常见的问题吧

首先，你会看到如果你下载SFML的与你的Visual C ++版本是对应的（一般都是对应的）
你可以完全忽略介绍中对于”自己构建SFML“的教程
因为那对于初学者确实有点困难

第二点，建议使用×64的版本
这点我想我就不用多说了
64位基本完全兼容32位
可以避免很多不必要的麻烦

第三点，关于初始的大绿球的代码

 #include <SFML/Graphics.hpp>   //不解释，目前只需要包含图像部分就行了。
               int main()                                 

{
 sf::RenderWindow window(sf::VideoMode(800, 600), "SFML Application");     //对新建窗口的封装，第一个参数是一个VideoMode，表示窗口大小，第二个表示窗口标题
 sf::CircleShape shape;                                                    //目前的物体，一个圆，暂时把他当成精灵就行了。
 shape.setPosition(400, 300);
 shape.setRadius(40.0f);
 shape.setFillColor(sf::Color::Cyan);

 while (window.isOpen()){                                                  //不断执行这个循环
 sf::Event event;

 while (window.pollEvent(event)){                                      //sfml中的事件，每次读取窗口的事件。
 if (event.type == sf::Event::Closed)                              //关闭事件，也就是按了窗口上的叉叉
  window.close();
 if (event.type == sf::Event::KeyPressed){                         //按键事件
 if (event.key.code == sf::Keyboard::Escape)                   //如果按了esc，也是关闭的效果
  window.close();
 }//end if
 }//end while

 window.clear();                                                       //每次清屏
 window.draw(shape);                                                   //东西还是要画的
 window.display();                                                     //画完之后还是要显示的
 }                                                                    //end while

 return 0;
 }


记得一定要把bin文件夹的dll文件放到exe文件的根目录里哟，我一般都放上（指不定用上哪个）

最后再说一个我最近碰到的小问题
在引用声效时
Failed to open sound file “resources/1.wav” (couldn’t open stream)
这是在VS里编译的时候DOS窗口出现的提示
不要慌，只是在VS里调试时候才会出现
直接从外部打开EXE文件就不会出现此提示了
个人认为是VS的访问权限及安全保护问题引发的
